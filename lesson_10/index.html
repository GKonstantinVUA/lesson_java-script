Загальна умова

Задача #0

Сформувати новий масив, у якому є тільки ті, що більші за 1000 грн

Задача #1

Сформувати новий масив, у якому є номери індексів тільки тих елементів, що більші за 1000 грн

Задача #2

Сформувати список з тих цін, які більші за попереднє значення

Задача #3

Задача #4

Підрахувати кількість змін цін

Задача #5

Задача #6

Визначати, чи усі ціни більше за 1000 грн

Задача #7

Підрахувати кількість цін, що більше за 1000 грн

Задача #8

Підрахувати суму цін, що більше за 1000 грн

Задача #9

Знайти першу ціну, що більше за 1000 грн

Задача #10

Знайти індекс першої ціни, що більше за 1000 грн

Задача #11

Знайти останню ціну, що більше за 1000 грн

Задача #12

Знайти індекс останньої ціни, що більше за 1000 грн
